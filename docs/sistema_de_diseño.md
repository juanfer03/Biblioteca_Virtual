# Sistema de diseno

## Principios visuales

- Claridad antes que decoracion.
- Sensacion de biblioteca fisica con capas, tarjetas y contrastes suaves.
- Componentes visuales reconocibles y faciles de migrar a React.

## Paleta base

- Fondo principal: tonos marfil y arena.
- Color de acento: cafe tostado.
- Superficies: blanco tibio con sombras suaves.
- Texto: gris oscuro con buen contraste.

## Tipografia

- Titulos: serif clasica para reforzar el caracter editorial.
- Texto y formularios: sans serif legible para la operacion diaria.

## UI atoms y patrones

- Botones redondeados con estado primario y secundario.
- Tarjetas para estadisticas y libros.
- Pildoras para filtros y estados.
- Formularios simples con jerarquia vertical.

## Linea de evolucion

El sistema de diseno debe mantenerse consistente en la futura migracion a React para no romper la identidad visual de la Fase 1.
