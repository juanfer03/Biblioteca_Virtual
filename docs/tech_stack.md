# Tech stack

## Stack obligatorio

- React.
- TypeScript.
- Open Library para catalogo publico gratuito.
- Firebase.
- CSS puro sin frameworks.

## Estado por fase

- Fase 1: HTML y CSS puros para la maqueta base.
- Fase 2: React + TypeScript con folder-by-component, Vite y Open Library para busqueda y alta local.
- Fase 3: Firebase Firestore para persistencia de libros.

## Decisiones tecnicas

- No usar frameworks de estilos para mantener el control visual y el alineamiento con el taller.
- Preparar componentes tipados desde el inicio de la migracion a React.
- Separar la fuente publica de catalogo de la autenticacion y persistencia privada.
- Separar documentacion y codigo de interfaz en artefactos distintos.
