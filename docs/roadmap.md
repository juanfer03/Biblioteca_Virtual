# Roadmap

## Fase 1 - Documentacion + maquetacion HTML/CSS

- Completar la documentacion central.
- Construir la maqueta estatica base.
- Definir sistema de diseno y arquitectura futura.

## Fase 2 - Migracion a React + TypeScript

- Crear estructura `src/components` por componente.
- Convertir la maqueta en componentes React.
- Implementar interactividad local.

## Fase 3 - Firebase

- Configurar Firestore.
- Persistir libros, estados y valoraciones.
- Preparar despliegue.

## Estado actual

La Fase 1 esta iniciada con documentacion central y una primera version estatica de la interfaz.
